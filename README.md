# Drawrr

Draw in the air with your hand. Built with Python and MediaPipe.

## How to Use

- Point index finger to draw
- Pinch thumb + index to toggle pen on/off
- Hold fist to clear canvas

## Run It

**Install:** pip install -r requirements.txt

**Run:**python server.py

Open http://localhost:8000

## Stack

Python | FastAPI | OpenCV | MediaPipe
